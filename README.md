# Brazilian-E-Commerce-Public-Dataset-by-Olist

📝 **Project Overview**

This project focuses on analyzing the Olist (Brazil) e-commerce dataset to gain insights into business performance, customer behavior, and operational efficiency. The data includes over 100,000 orders from 2016 to 2018.

🔍 **Main Objectives**

- Tracking revenue trends and growth over time

- Customer segmentation (RFM Analysis) to optimize marketing strategies

- Analyzing the correlation between delivery speed and customer satisfaction (Review Score)

🛠 **Tech Stack**
- Data cleaning and preparation: Python, Pandas

- Database: SQL - Data processing and extraction

- Visualization: Tableau - Building interactive dashboards

- Documentation: Notion & GitHub
