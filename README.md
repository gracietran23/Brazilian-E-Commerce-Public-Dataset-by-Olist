# 🛒 Brazilian E-Commerce Public Dataset by Olist

## 📌 Project Overview

This project analyzes the Olist Brazilian E-Commerce dataset to understand business performance, customer behavior, delivery efficiency, and seller performance.

The goal is to transform raw transactional data into meaningful insights that can support business decision-making.



## ⭐ Objectives

* Monitor key business KPIs
* Analyze revenue trends and customer behavior
* Evaluate operational efficiency through delivery data and customer feedback
* Visualize data using dashboards to support decision-making


## ⚙️ Project Workflow

### 1. Data Preparation

* Checked missing values
* Validated data consistency
* Converted data types

### 2. SQL Analysis

Business questions:

* How is revenue changing over time?
* Which product categories generate the most revenue?
* Which regions contribute the most sales?
* Who are the most valuable customers?
* How does delivery performance affect customer satisfaction?
* Which sellers receive the most negative reviews?

### 3. Dashboard Development

Created interactive Tableau dashboards to monitor key business metrics and visualize insights.


## 💡 Key Analysis

### 1. Sales Performance

* Monthly revenue trend
* Revenue growth rate
* Top product categories
* Revenue by state
* Peak purchasing hours

### 2. Customer Analysis

* Customer value (RFM metrics)
* Customer retention rate

### 3. Delivery Analysis

* Delivery delay rate
* Delivery performance vs review score

### 4. Seller Analysis

* Seller review performance
* Seller contribution to low-rated orders



## 🛠️ Tech Stack

* **Dataset**: [Brazilian E-Commerce Public Dataset by Olist](https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce)
* **Database:** SQL Server
* **Visualization:** Tableau
* **Language:** Python (`pandas`, `matplotlib`, `seaborn`)
